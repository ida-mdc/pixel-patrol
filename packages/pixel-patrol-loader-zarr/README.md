# PixelPatrol Zarr Loader Extension (`pixel-patrol-loader-zarr`)

This is a lightweight extension for **PixelPatrol** that loads **Zarr and OME-Zarr** stores — no bioio dependency required.

## Supported formats

- `.zarr` — Zarr array or group
- `.ome.zarr` — OME-NGFF (axes from multiscales metadata)

## Installation

```bash
uv pip install pixel-patrol-loader-zarr
```

Or as part of the full stack:

```bash
uv pip install pixel-patrol
```
