Understand your image dataset

Image datasets are rarely as clean or consistent as they appear. Pixel Patrol analyses your images and generates a shareable, interactive browser report - file and image metadata, pixel statistics, quality metrics, and per-dimension slice statistics. Get immediate results, compare conditions, catch outliers, verify batch consistency, and get the full picture before you use your dataset.

Your data stays local - nothing is uploaded.

Our automated pipeline

Four steps from folder to insight.

Point Pixel Patrol to any folder of images. It supports common formats and walks subdirectories automatically, so no manual file listing is needed.

Images are scanned in parallel using configurable plugins that extract pixel statistics, quality metrics, and file metadata in one pass.

All results are written to a compact Parquet file stored alongside your data, ready to share or load into other tools.

Open the report in any browser — no server needed. Group by dimension, filter outliers, compare conditions, and inspect individual images.
